# ctf talk

## Browser Slideshow

`cd` and open `docs/index.html` or visit https://alexflor.es/ctf-talk

## Terminal Slideshow
Slides are in talk.yml

To run the presentation:

```bash
git clone https://github.com/audibleblink/ctf-talk.git
cd ctf-talk
bundle install
bundle exec termnote talk.yml
```
